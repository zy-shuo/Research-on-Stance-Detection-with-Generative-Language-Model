# 微博立场检测数据集 (Weibo Stance Detection Dataset)

## 📋 数据集概述

本数据集是专门用于中文社交媒体立场检测任务的标注数据集，数据来源于微博平台的真实用户评论。数据集涵盖了五个具有广泛社会关注度的热点事件：

1. **唐山打人事件**
2. **胡鑫宇失踪事件**
3. **女子不让6岁男童上女厕所遭痛骂事件**
4. **满江红起诉大V事件**
5. **女子泼水节被围着泼水撕雨衣事件**

## 📊 数据统计

### 版本 v1
- **总数据量**: 2,500 条
- **训练集**: 2,000 条（每个事件 400 条）
- **测试集**: 500 条（每个事件 100 条）

### 版本 v2
- **总数据量**: 1,698 条
- **改进说明**: 进一步提升了数据质量，经过更严格的标注审核和数据清洗


## 🏷️ 数据字段说明

每条数据包含以下主要字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `label` | int | 立场标签：`0` - 支持，`1` - 反对，`2` - 中立 |
| `comment` | str | 用户评论文本内容 |
| `target` | str | 立场目标（事件主题） |
